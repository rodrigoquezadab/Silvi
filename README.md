# Silvi - Catálogo Digital Textil y Hogar

Catálogo interactivo desarrollado para la tienda **Textil y Hogar**. Este sitio web sirve de vitrina digital para los productos de ropa interior y ropa de trabajo que se comercializan en la tienda física de manera presencial.

## 🚀 Tecnologías Utilizadas

- **HTML5**
- **JavaScript (Vanilla)**
- **Tailwind CSS v3.4**
- **CSS3 Personalizado**

## 📂 Estructura del Proyecto

- `index.html` - Interfaz principal del catálogo y filtros.
- `app.js` - Base de datos local de productos y lógica interactiva.
- `styles.css` - Estilos adicionales para glassmorphism y animaciones.
- `contexto.md` - Documentación detallada del funcionamiento, características y cambios.
- `Media/` - Carpeta con las imágenes de los productos y diagramas de medidas.
- `Documentos/` - Archivos originales con especificaciones y precios.

## 👥 Cómo Funciona la Compra
1. El cliente explora el catálogo digital en línea.
2. Filtra por tipo de prenda (ropa interior, delantales de trabajo), público o materiales.
3. Al ver los detalles de un producto en el modal, puede pulsar el botón **"Consultar Disponibilidad"** para abrir un chat directo de WhatsApp con la tienda física, con un mensaje pre-redactado de interés.
4. El retiro y pago de las prendas se realiza de forma **presencial** en la tienda.
